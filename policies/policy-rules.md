# 입코딩 규칙

## 입코딩 시작 전 60분

- 키보드 및 마우스 사용을 전면 허용합니다.
- 마이크 및 헤드셋 설정을 점검하세요.
- 마우스/트랙패드로 음성 입력을 할 수 있도록 설정하세요.
- 필요한 모든 VS Code 설정, GitHub Copilot 설정, GitHub Copilot CLI 설정, MCP 서버 설정 등을 마치세요.
- `AGENTS.md`과 Skills 등을 포함한 필요한 모든 Custom Instructions 도 설정해 두세요.
- `PRD.md` 문서를 포함해서 필요한 모든 문서들을 준비해 두세요.

## 입코딩 중

- VS Code와 GitHub Copilot, Copilot CLI만 허용합니다. 다른 개발 도구는 사용할 수 없습니다.
- 키보드 위에 수건을 덮어둡니다.
- 키보드와 마우스/트랙패드는 **절대로** 사용할 수 없습니다.
- 키보드와 마우스/트랙패드 사용 예외:
  - 음성 입력을 위해 마우스/트랙패드를 사용할 수 있습니다.
  - 앱 제출을 위해 키보드와 마우스/트랙패드 사용을 승인 받을 수 있습니다.
  - 컴퓨터 사용에 문제가 생겼을 경우 문제 해결을 위해 키보드와 마우스/트랙패드 사용을 승인 받을 수 있습니다.
- 개인 마이크를 통해 음성으로 코딩합니다.
- 부적절한 행위 적발시 경고(옐로우 카드), 죄인존 퇴장(레드 카드), 이벤트 아웃(강제 퇴장)의 페널티가 있습니다. [페널티 규정](./policy-penalties.md) 문서를 참조하세요.

## 입코딩 종료 30분 전

- 키보드 및 마우스 사용을 전면 허용합니다.
- 입코딩 종료 시점까지 앱 제출을 하지 못할 경우 자동으로 탈락합니다.
